Trade-offs
Variant A
Larger dot + outer ring
+ Unmistakeable hierarchy — ring reads at distance
+ No transform conflict — just size + box-shadow
+ Scales cleanly with existing hover logic
– Slightly more visual weight in dense graphs
Variant B
Rounded square
+ Clear shape differentiation (circle ≠ square)
+ Low extra weight vs Variant A
– Subtle at small sizes — 14px may not read strongly
– Less conventional for graph node hierarchies
Variant C
Hollow ring
+ Elegant — hollow = container, filled = task
+ Mental model matches "issue that holds others"
– Conflicts with blocked style (also hollow + dashed border)
– Low contrast against dark background

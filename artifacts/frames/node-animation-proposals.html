dev_state = "idle"

1. Idle

No branch linked, no open PR. Default rendering — unchanged from existing behavior.

dev_state = "dev"

2. Dev started

≥1 branch matches ^(?:[a-z]+/)?{N}-. Halo pulses outward (1.6s) signaling active local work. Auto-cleared on branch delete.

dev_state = "pr_open"

3. PR open

Open PR linked (no reviewed label). Pulse + 1 satellite dot orbiting the node (1.8s). Work pushed, awaiting review.

dev_state = "pr_reviewed"

4. PR reviewed

PR has reviewed label. Pulse + 2 satellite dots (180° apart, magenta accent). Auto-merge armed, waiting for CI to merge.

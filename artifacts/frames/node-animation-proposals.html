Node Animation Proposals — In-Progress / Paired States

dep-graph v6 · 3 options to pick from · animations visible but non-distracting

in-progress (solo) — cyan accent paired (à 2) — magenta secondary

01 Halo Pulse

A single ring expands outward from the node and fades. Calm, breathing rhythm (1.6s). For paired tasks, a second magenta halo is staggered by 0.8s — reads as two heartbeats overlapping. 

/* in-progress */
.gg-node.in-progress::before {
  content: '';
  position: absolute;
  inset: -2px;
  border-radius: 50%;
  border: 1.5px solid var(--accent);
  animation: halo 1.6s ease-out infinite;
}
/* paired: add ::after with --pair color
   and animation-delay: 0.8s */

@keyframes halo {
  0%   { transform: scale(.85); opacity: .9; }
  80%  {                         opacity: 0;  }
  100% { transform: scale(2.6); opacity: 0;  }
}

02 Rotating Dashed Ring

A dashed ring rotates around the node (2.2s). Distinct silhouette versus the static .blocked dashed border. Paired = two concentric rings spinning in opposite directions, different colors. 

/* in-progress */
.gg-node.in-progress::before {
  content: '';
  position: absolute;
  inset: -5px;
  border-radius: 50%;
  border: 1.25px dashed var(--accent);
  animation: spin-cw 2.2s linear infinite;
}
/* paired: ::before (cw, cyan) + ::after
   (ccw, --pair) at inset:-9px */

@keyframes spin-cw  { to { transform: rotate(360deg);  } }
@keyframes spin-ccw { to { transform: rotate(-360deg); } }

03 Orbiting Dot

A small satellite dot orbits the node (1.8s). Suggests "active work circling". For paired, two dots orbit at 180° apart with different colors — visually reads as "two people in motion around the issue". 

/* in-progress: 1 orbiting satellite */
.gg-node.in-progress::before {
  content: '';
  position: absolute;
  inset: -6px;
  border-radius: 50%;
  background: radial-gradient(
    circle at 50% 0%,
    var(--accent) 0 2.2px,
    transparent 2.6px);
  animation: orbit 1.8s linear infinite;
}
/* paired: ::after with --pair gradient
   + animation-delay: -0.9s (180° offset) */

@keyframes orbit { to { transform: rotate(360deg); } }

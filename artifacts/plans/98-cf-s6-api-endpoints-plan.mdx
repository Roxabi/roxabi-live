## Micro-Tasks

### T1 — parse.ts (`worker/src/sync/parse.ts`)
Port `parse_milestone`. Regex parity: `^M(\d+)\s*[—–-]\s*(.+)$`→`{M{n},name,n}`;
`^Phase\s+(\d+)\s*[—–-]\s*(.+)$`→`{Ph{n},name,100+n}`; `Final Initiatives`→`{FIN,…,999}`;
`null`→`{null,null,1000}`; else→`{null,raw,1000}`. Verify: `npm test parse`.

### T2 — graph.ts (`worker/src/api/graph.ts`) [blockedBy T1]
Port `build_graph_json`+`_compute_dev_state`+`_lane_from_labels`. 4 D1 reads
(labels grouped, open pr_state JSON-parsed, issues full, edges). Node/Edge shape per
`api.py`. Wire `GET /api/graph`. Verify: response keys == Python `{nodes,edges}`.

### T3 — issues.ts (`worker/src/api/issues.ts`)
Port `list_issues` (filters repo/state/label EXISTS/limit/offset, ORDER updated_at ASC,
`{issues,total,limit,offset}`) + `get_issue` (`ISSUE_KEY_RE`→400, 404, blocking/blocked_by
edge arrays w/ stub fallback). Wire `GET /api/issues` + `GET /api/issues/:key`.

### T4 — admin.ts (`worker/src/api/admin.ts`)
`POST /admin/sync` → `c.executionCtx.waitUntil(runSync(env))` → `202 {ok:true,triggered:true}`.
OTP at CF Access edge (defer open-Q7 JWT). Wire route; fix stale router comment.

### T5 — tests [blockedBy T2,T3,T4]
parse/graph/issues/admin `.test.ts` mirroring `sync.test.ts` style. Verify: `npm test`.

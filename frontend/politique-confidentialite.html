Politique de confidentialité

Dernière mise à jour : 19 juin 2026

La présente politique décrit comment Roxabi Live (live.roxabi.dev) traite les données personnelles dans le cadre de son service de visualisation de graphes de dépendances GitHub.

1. Responsable du traitement

Mickael Bouly — contact : mickael@bouly.io

2. Données traitées

  • Identité GitHub : identifiant, login, avatar (via OAuth).
  • Données d'organisation : métadonnées de dépôts, issues (état, labels, relations, milestones), synchronisées via l'application GitHub.
  • Contenu chiffré : titres et corps d'issues chiffrés côté client avant stockage ; Roxabi Live ne conserve pas ces contenus en clair sur le serveur.
  • Données techniques : journaux d'exécution du service (horodatage, statut des requêtes), cookies de session.

3. Finalités et bases légales

  • Fourniture du service (exécution du contrat / intérêt légitime) : authentification, affichage du graphe, synchronisation GitHub.
  • Sécurité (intérêt légitime) : protection des comptes, détection d'abus, intégrité de la plateforme.
  • Support (intérêt légitime) : réponse aux demandes envoyées à l'éditeur.

4. Durée de conservation

  • Session utilisateur : 8 heures (cookie), révocable via déconnexion.
  • Données de corpus (issues, graphe) : conservées tant que le compte / l'installation GitHub est active ; suppression sur demande ou retrait de l'application.
  • Journaux techniques : durée limitée (quelques jours à quelques semaines selon l'infrastructure).

5. Sous-traitants

Les données peuvent être traitées par :

  • Cloudflare, Inc. — hébergement Worker, base D1, stockage R2 (audit).
  • GitHub, Inc. — authentification OAuth et source des données issues (sous votre contrôle via l'installation de l'application).

6. Transferts hors UE

Cloudflare et GitHub peuvent traiter des données aux États-Unis. Ces prestataires appliquent des garanties contractuelles (clauses types, certifications) conformes au RGPD.

7. Vos droits

Conformément au RGPD, vous disposez des droits d'accès, de rectification, d'effacement, de limitation, d'opposition et de portabilité. Pour les exercer : mickael@bouly.io. Vous pouvez introduire une réclamation auprès de la CNIL (www.cnil.fr).

8. Cookies

Roxabi Live utilise un cookie strictement nécessaire (__Host-session) pour maintenir votre session après connexion GitHub. Aucun cookie publicitaire n'est déposé. Les préférences d'interface (thème, consentement) peuvent être stockées localement dans votre navigateur (localStorage).
